### Hướng dẫn run code:
-	Bước 1: Mở terminal gõ “npm I” để cài các module cần thiết
-	Bước 2: gõ “npm start” để chạy website
-	Bước 3: mở trình duyệt truy cập “localhost:3000” để xem kết quả
